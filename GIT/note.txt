mở terminal: ctrl + shift + `
câu lệnh git cơ bản
repository: (kho chứa dữ liệu của git), 2 loại

1. local repo: ở trên máy
2. remot repo: kho online

file trong máy | vùng nhớ tạm | local repo | remote repo

git init:tạo repository:ở local
git add .:đẩy tất cả các file vào trong một vùng nhớ
git commit -m "...": đẩy từ vùng nhớ tạm t hời vào local repo, có nhiệm vụ là đánh dấu mốc lịch sử tại đây
git remote... : Kết nối local repo với remote repo
git push origin master: Đẩy những thay đổi từ local lên remote
Lưu ý Từ lần thứ với dự án này: git add , git commit, git push
git clone: copy dự án từ remote về local
git pull: kéo dữ liệu mới nhất về dự án
